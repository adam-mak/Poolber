# Poolber
Poolber is a user-oriented carpooling app that encourages rider to share rides together. This not only helps reduce the gaseous emissions and traffic congestion, but it also helps you cut down the cost of your ride (and maybe meet new people!). Feel free to dive down below for a demo of the app. If you want to run the project locally, feel free to get started with our [Development Steps](https://github.com/adam-mak/SE3A04/blob/main/D4/CarpoolProjectExpo/README.md)! Note that you will need to configure your own Google Maps API Key and Firebase API Key.
